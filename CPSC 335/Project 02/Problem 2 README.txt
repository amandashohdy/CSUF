Problem 2: Arraylist and Subsets
================================

Overview:
This algorithm determines the maximum sum of a subarray within an array of integers. 
This program accepts an array of integers and outputs the subarray that results in the
maximum sum.

Running the Program:
Navigate to the directory containing the program files and start the program. 
Enter the desired integers with spaces in between. When finished entering values, press the
enter key. 

===============
Example Output:
===============

Enter a list of integers separated by spaces: -3 -5 5 -1 -3 1 5 -6
User input list: [-3, -5, 5, -1, -3, 1, 5, -6]
Largest sum subarray: [5, -1, -3, 1, 5]
