Problem 1: Ensuring Convenient Schedules
========================================

Overview: 
This algorithm determines the blocks of time team members are free to have a meeting of a specified duration.
This program accepts vectors of blocks of time each member is busy, a vector containing their login and 
logout times, and the duration of the meeting. This program outputs a series of available meeting times for 
the members. 

Running the Program:
Navigate to the directory containing the program files and start the program.
