Algorithm 1: Connecting Pairs of Persons User Guide 
===================================================

Overview:
This algorithim allows users to connect pairs of people based on index. 
The program accepts integers representing individuals and stops accepting input when it encounters -1. 
The number of individuals should be less than or equal to 30 and even. 

Running the Program:
Navigate to the directory containing the program files and start the program.
Enter desired individuals with spaces in between.
Enter -1 to stop inputting individuals. 

===============

===============

Enter -1 when finished inputing numbers...
0 2 1 3 -1

0 2 1 3
0 1 2 3
